# SPIN LAMB!

一言で言うと：3D押し出しアクションゲーム

## 概要

四角形の草原のフィールドを舞台に、羊のキャラクター同士が突進し、ぶつかり合い、相手をトゲのついた柵に当てることが目的の3Dアクションゲームです。
その場でボタンを長押ししてパワーを溜め、解放することで、より長い距離を移動し、遠くまで吹き飛ばすことができます。

1分間の制限時間の中で相手を柵に当てた回数を競い合う「バトル」と、時間無制限に操作できる「フリープレイ」の二種類の遊び方があります。
また、2人プレイに対応しています。(2人プレイにはXBoxコントローラーが2つ必要です)

## 開発環境

-  VisualStudio2022
-  DirectX11
  
## 必要ライブラリ

-  FBX SDK

## 動作環境

- PCのみ対応

## 操作方法

### キーボード操作

- `矢印キー` :  移動
- `スペースキー` : ジャンプ
- `シフトキー長押し` : チャージ
- `シフトキーを離す` : ダッシュ攻撃
- `チャージ中にスペースキー` : チャージ状態を解除
- `WASDキー` : カメラ移動
- `Zキー` : カメラリセット
- `ESCキー` : ポーズ画面を開く

### コントローラー操作

- `左スティック` : 移動
- `Aボタン` : ジャンプ
- `Bボタン長押し` : チャージ
- `Bボタンを離す` : ダッシュ攻撃
- `チャージ中にAボタン` : チャージ状態を解除
- `右スティック` : カメラ移動
- `X・Yボタン` : カメラリセット
- `STARTボタン` : ポーズ画面を開く

## 開発経緯

これまでに公開したゲーム作品の感想にて「操作が難しい」「爽快感が欲しい」と行った意見・反応をいただきました。

その反省から、手触りが良いこと・視覚的にわかりやすいこと・短時間でも楽しめることを意識し、この作品を制作しました。
